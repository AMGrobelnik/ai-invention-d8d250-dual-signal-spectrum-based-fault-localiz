## Layman Summary

Tests five AI reasoning methods on 150 logic puzzles, measuring how accurately each proves or disproves conclusions from given facts, with bootstrap confidence intervals and effect-size statistics.

## Full Summary

This evaluation artifact implements a complete end-to-end statistical comparison of five Prolog-based and neural reasoning methods on 150 examples from the ProofWriter logical reasoning benchmark (used as a publicly accessible substitute for the gated FOLIO dataset, which has identical structure: NL premises, NL conclusion, True/False/Unknown label, and explicit hop-depth annotations via QDep field).

## Methods Evaluated
1. **dual_sbfl** — Full pipeline: LLM extracts Prolog KB, LLM oracle generates test queries, stratified SLD trace (tracked via a pure-Python Prolog interpreter with direct/indirect fault distinction), Ochiai scores computed with stratification, sub-goal harvesting identifies missing predicates, top-3 suspicious predicates repaired over 2 rounds.
2. **one_shot** — Single LLM call to extract KB, no repair. Baseline.
3. **cot** — Chain-of-thought: LLM reasons step-by-step and answers ENTAILED/NOT_ENTAILED/UNKNOWN directly, no Prolog KB.
4. **self_refine** — KB extraction + up to 2 rounds of LLM re-generation with global pass-rate feedback only (no localization).
5. **flat_sbfl** — Same as dual_sbfl but using flat binary Ochiai (no stratification into direct/indirect fault, no sub-goal harvesting). Ablation study.

## Primary Results (N=150, ProofWriter validation)
- **dual_sbfl**: accuracy=0.193, hallucination_rate=0.016, mean_llm_calls=7.0
- **one_shot**: accuracy=0.200, hallucination_rate=0.012, mean_llm_calls=2.0
- **cot**: accuracy=0.320, hallucination_rate=1.000 (no KB), mean_llm_calls=1.0
- **self_refine**: accuracy=0.187, hallucination_rate=0.028, mean_llm_calls=5.0
- **flat_sbfl**: accuracy=0.187, hallucination_rate=0.021, mean_llm_calls=7.4

## Statistical Tests
- **Bootstrap CIs** (10,000 resamples): dual_sbfl vs one_shot CI=[-0.073, 0.060] — includes zero, no significant improvement.
- **Cohen's h** (dual_sbfl vs one_shot): h=-0.017 — below 0.2 threshold, negligible effect size.
- **McNemar test** (dual_sbfl vs cot): p=0.0005 — significant; CoT significantly outperforms Prolog-based approaches. All other Prolog method pairs: p≈1.0 (no significant difference).
- **Oracle calibration Spearman ρ**: -0.167 — BELOW the pre-registered threshold of ρ≥0.6. This triggers the pre-registered invalidation criterion: the LLM oracle does not provide sufficient localization signal for SBFL-based repair.

## Hypothesis Assessment
The hypothesis that dual-signal SBFL improves multi-hop deduction accuracy is **disconfirmed** on this benchmark. Key findings: (1) CoT significantly outperforms all Prolog-based methods; (2) SBFL repair does not improve over one-shot KB extraction; (3) Oracle calibration fails (ρ<0.6), meaning the localization assumption underlying SBFL is violated; (4) Hallucination reduction is marginal (dual_sbfl reduces vs self_refine but not vs one_shot).

## Technical Details
- Dataset: tasksource/proofwriter (validation split, 150 examples balanced across QDep 0–3+ hop depths)

- Pure-Python Prolog interpreter implemented with call-count limit (2000 calls/query) to prevent exponential backtracking
