## Dataset Artifact: FOLIO + ProofWriter-NL for Dual-Signal SBFL Evaluation

### Datasets Included

**Dataset 1 — FOLIO (folio)**
- Source: HannaAbiAkl/FOLIO-KR on HuggingFace (mirrors gated yale-nlp/FOLIO; same data with additional KR notations)
- Paper: Han et al., EMNLP 2024, arXiv:2209.00840
- 1,204 examples across train (800) / val (183) / calibration (20) / test (201) splits
- Each example: NL premises (concatenated, ≤3000 chars), NL conclusion, gold label (True/False/Unknown), gold FOL premises (list of strings), gold FOL conclusion (string)
- Label distribution: True=460, False=351, Unknown=393
- Calibration subset: first 20 validation examples marked is_calibration_doc=true, split='calibration'
- Metadata fields: metadata_fold, metadata_split, metadata_is_calibration_doc, metadata_gold_label, metadata_premises_text, metadata_conclusion, metadata_gold_fol_premises (JSON-encoded list), metadata_gold_fol_conclusion, metadata_story_id, metadata_example_id, metadata_task_type, metadata_fold_name

**Dataset 2 — ProofWriter-NL RuleTaker depth-3 (proofwriter_ruletaker_depth3)**
- Source: tasksource/ruletaker on HuggingFace, config=depth-3ext-NatLang
- Paper: Clark et al., IJCAI 2020 (RuleTaker); Tafjord et al., ACL Findings 2021 (ProofWriter)
- 300 balanced examples (150 True / 150 False), all split='test'
- Each example: NL context (facts + rules ≤3000 chars), NL question as conclusion, True/False gold label
- No FOL annotations (gold_fol_premises=[], gold_fol_conclusion='')
- Uses depth-3ext-NatLang config: natural-language surface forms requiring 3-hop deduction chains

### Schema
All examples conform to exp_sel_data_out schema:
- input: "Premises: {premises_text}\nConclusion: {conclusion}"
- output: gold label string ("True", "False", or "Unknown")
- metadata_* fields for all provenance and annotation data
